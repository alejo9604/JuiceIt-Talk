## Shortcuts
- **P**: Next Juice.
- **O**: Prev Juice.
- **N**: Add low Trauma.
- **M**: Add high Trauma.
- **R**: Reset Background Tiles.
- **T**: Toggle Slowmo.
- **Esc**: Config window (All Juice effects).
- **Shift + 1**: Toggle Trauma UI.
- **Shift + 2**: Toggle enemies spawn.
- **Shift + 3**: Reset Player health.
- **Shift + 4**: Toggle Player movement.
- **Shift + 5**: Toggle Camera lines.
